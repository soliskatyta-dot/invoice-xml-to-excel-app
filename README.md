#  Invoice XML to Excel App

##  Description
This project is a web application that allows users to:
- Upload invoice XML files
- Extract structured invoice data
- Export structured data into Excel format

##  Technologies Used
- HTML
- JavaScript
- Bootstrap
- XLSX library

##  Scrum Implementation
This project is managed using Scrum methodology:
- 2-week sprints
- Product Backlog
- Sprint Planning
- Daily tracking
- Sprint Retrospectives

## 🎯 Project Goal
To automate invoice data management and improve reporting efficiency.
